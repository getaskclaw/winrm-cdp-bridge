## If access breaks

On the Windows VPS, run as Administrator:

```powershell
net user winrm_user "NEW_KEEPASS_PASSWORD"
net localgroup Administrators winrm_user /add
Enable-PSRemoting -Force
Restart-Service WinRM
```

Then update `./2604/26429` with the new password.

## Cleanup when done

If Kestrel no longer needs access:

```powershell
net user winrm_user /delete
Remove-ItemProperty -Path HKLM:\SOFTWARE\Microsoft\Windows\CurrentVersion\Policies\System `
  -Name LocalAccountTokenFilterPolicy
Restart-Service WinRM
```
